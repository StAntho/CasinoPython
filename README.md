## Installation

Install Python version `3.10.6`

Install package dotenv and mysql-connector :
```bash
pip install python-dotenv
pip install mysql-connector-python
```

Create your file `.env` with :
```
HOST_NAME
DATABASE_NAME
USER_NAME
PASSWORD
```

## Run Locally
Execute `start.bat`
